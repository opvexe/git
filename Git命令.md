### Git 提交但未push，返回提交状态操作
```
git log 

# 保留本次提交
git reset 624f7521ca24e123dadd10665e99314c690d4351 (非本次提交的commitId)

# 不保留本次提交
git reset -hard 624f7521ca24e123dadd10665e99314c690d4351
```
